Basics

What Is Arduino :

Arduino is an open-source electronics platform based on easy-to-use hardware and software. Arduino boards are able to read inputs and perform them.The inputs to arduino boards are given through Arduino IDE.The Arduino programming language is based on a very simple hardware programming language called processing, which is similar to the C language. After the sketch is written in the Arduino IDE, it should be uploaded on the Arduino board for execution.The open source Arduino IDE runs on Windows, Mac OS X, and Linux.

Boards

There are several boards which can be programmable by using arduino IDE,some of them are given below. UNO board is recommended for beginners.Let us learn some terms before reading boards description.
Microcontroller : it is brain of each board which runs the program and send commands to different components through digital and analog pins.
RAM : The random access memory available to run code.We should not write a program which occupies higher memory than RAM of board.
operating voltage : Max Output voltage of pins of board.voltage applied to board through USB or external battery should be greater than or equal to operating voltage. These boards have a inbuild voltage regulator which suplies 5V even though we apply high voltage upto certain extent.
Digital I/O pins : These pins reads or writes high or low voltages.High voltage refers to operating voltage,low voltage refers to zero voltage.these pins cant read or write intermediate voltages.
Analog pins : used to read analog inputs

Arduino UNO:

  1. Dimension :2.7 in*2.1 in
  2. Processor :Atmega328P
  3. operating Voltage :5V
  4. RAM :2KB
  5. Digital I/O pins :14
  6. Digital I/O pins with PWM :6
  7. Analog pins :6
  8. USB connectivity :standard A/B usb
  9. Flash memory(KB) :32
  10. Wi-fi :No(module can enable wifi to uno)
  11. Bluetooth :No(module can enable Bluetooth to uno)

Arduino Mega :

  1. Dimension :4 in*2.1 in
  2. Processor :ATmega2560
  3. operating Voltage :5V
  4. RAM :2KB
  5. Digital I/O pins :54
  6. Digital I/O pins with PWM :15
  7. Analog pins :16
  8. USB connectivity :standard A/B usb
  9. Flash memory(KB) :256
  10. Wi-fi :No(module can enable wifi to mega)
  11. Bluetooth :No(module can enable Bluetooth to mega)

Arduino micro :

  1. Dimension :0.7 in*1.9 in
  2. Processor :ATmega32U4
  3. operating Voltage :5V
  4. RAM :2KB
  5. Digital I/O pins :20
  6. Digital I/O pins with PWM :7
  7. Analog pins :12
  8. USB connectivity :Micro-USB
  9. Flash memory(KB) :32
  10. Wi-fi :No
  11. Bluetooth :No

ESP32 :

  1. Processor :Tensilica Xtensa LX6 microprocessor
  2. operating Voltage :3.3V
  3. RAM :512KB
  4. Digital I/O pins :36
  5. Digital I/O pins with PWM :16
  6. Analog pins :18
  7. USB connectivity :Micro-USB
  8. Flash memory(MB) :16
  9. Wi-fi :available on chip
  10. Bluetooth :available on chip

Breadboard

To make electronic circuits we use breadboard instead of soldering. components and wires can be reusable if we use breadboard.

  • Vertical 5 holes corresponding to A,B,C,D,E in figure are at same potential.i.e;connected by a metal chip.
  • All holes beside +voltage are at same potential
  • All holes beside _voltage are at same potential

LED's

long terminal is cathode and short terminal is anode

Tutorial 1 :

  • open IDE and select board and COM
  • connect led cathode to pin 13 and anode to Gnd
  • now write code as follows
    #define led 13
  • declare led pin as output
    void setup(){
    pinMode(led,OUTPUT);}
  • blink led for every 1 second
    void loop(){
    digitalWrite(led,HIGH);
    delay(1000); //wait 1000 milli seconds
    digitalWrite(led,LOW);
    delay(1000); //wait 1000 milli seconds}

potentiometer

it is used to obtain intermediate voltage between zero and operating voltage by rotating knob.connect 2 end terminals to high and low voltage.connect middle tetminal to any analog pin of UNO say A0 .

CODE:
#define potentiometer A0
void setup(){
Serial.begin(9600); // initialize serial communication at 9600 bits per second
pinMode(potentiometer,INPUT);
int sensorValue = analogRead(A0); // read the input on analog pin 0:
float voltage = sensorValue * (5.0 / 1023.0);// Convert the analog reading (which goes from 0 - 1023) to a voltage (0 - 5V):
Serial.println(voltage);// print out the value you read: }

Sensors :

Temperature

LM35 is a temperature sensor.
CODE :
float temp;
int tempPin = 0;
void setup() {
Serial.begin(9600); }
void loop() {
temp = analogRead(tempPin); // read analog volt from sensor and save to variable temp
temp = temp * 0.48828125; // convert the analog volt to its temperature equivalent
Serial.print("TEMPERATURE = ");
Serial.print(temp); // display temperature value
Serial.print("*C");
Serial.println();
delay(1000); // update sensor reading each one second }
